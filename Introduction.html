🎯 项目背景与目标

金融投资领域每天产生海量数据，分析师需要手动查询行情、计算技术指标、评估风险并撰写投资备忘录——耗时且易错。本项目利用 .NET 10 + Semantic Kernel + ML.NET 构建一个能够可靠完成投资相关任务的 AI Agent，实现数据查询、技术分析、异常检测、风险聚类及智能备忘录生成的全流程自动化。

✔️ 查询股票数据（价格、成交量、持仓）
✔️ 计算移动平均线(5/10/20/30/60/120)、RSI、年化波动率
✔️ 基于统计突变的价格异常检测
✔️ K‑Means 风险等级分类（低/中/高）
✔️ ML.NET 回归预测次日收盘价
✔️ ONNX 模型情感/风险分析（可扩展）
✔️ 大语言模型生成自然语言投资建议
✔️ 企业级可靠性：网关、限流熔断、结构化日志、单元测试

⚙️ 核心功能模块

📊 股票数据服务

Repository 模式 + HttpClient + Polly 弹性策略，从 API 或模拟源获取股票实时/历史数据。

📐 技术指标计算

MA5~MA120、RSI、年化波动率。纯数学算法 + ML.NET 辅助，精度高。

⚠️ 价格异常检测

基于 ML.NET TimeSeries 的 IID 突变点检测，识别价格模式持久性改变。

🔖 风险聚类

K‑Means 聚类（波动率、RSI、夏普比率、最大回撤）输出低/中/高风险等级。

📈 价格预测

利用历史5日价格+成交量，使用 SDCA 回归预测次日收盘价。

🧠 AI 备忘录生成

Semantic Kernel 调用 LLM（OpenAI/智谱），基于技术指标和风险分生成专业投资建议。

🛡️ 弹性与可靠性

Polly 重试、熔断、限流、舱壁隔离；YARP 网关统一入口。

🏗️ 系统架构

[客户端/浏览器] → YARP Gateway (:5000) → AgentService (:5001) │ │ │ ├─ MemoController │ ├─ StockService (插件) │ └─ MemoService (插件) ↓ ↓ [弹性策略] TechnicalIndicatorService (限流/熔断/重试) ├─ ML.NET 回归/聚类/异常检测 └─ OnnxPredictionService └─ IStockRepository (HTTP + Polly)

🔄 典型工作流：生成投资备忘录

  1. 客户端 POST /api/agent/GenerateMemo (股票代码+日期)
  2. YARP 网关路由到 MemoController
  3. StockService.GetStockAsync 调用 Repository 获取股票数据（自动重试/熔断）
  4. TechnicalIndicatorService.CalculateIndicators 计算 MA、RSI、波动率；数据足够时执行 ML.NET 预测、异常检测、风险聚类
  5. MemoService.GenerateMemoAsync 计算风险分 (0~100) + 调用 LLM (Semantic Kernel) 生成自然语言投资建议
  6. 返回结构化 InvestmentMemo JSON，Serilog 记录完整请求上下文

🛡️ 企业级可靠性设计

⏱️ 重试与熔断

指数退避重试(最多3次) + 断路器(5次失败触发，30秒半开探测)，防止下游雪崩。

🚦 限流与舱壁

令牌桶限流 (100次/分钟/IP)；舱壁隔离限制最大并发10，避免资源耗尽。

📋 结构化日志

Serilog 输出到控制台和滚动文件，自动丰富机器名/线程Id/请求TraceId，便于 ELK 接入。

🧪 测试与评估

xUnit 单元测试覆盖服务层逻辑，Moq 模拟依赖；LangSmith 评估脚本验证 Agent 输出质量。
